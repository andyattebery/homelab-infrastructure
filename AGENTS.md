
## Ansible

- The ansible directory contains ansible playbooks and roles to provision and configure mostly homelab servers

### Tools

- [mise](https://mise.jdx.dev) manages
  - [python](https://mise.jdx.dev/lang/python.html) in `ansible/.venv`
  - [tasks](https://mise.jdx.dev/tasks/) to bootstrap and update the ansible environment
- System tools in $PATH
  - gh
  - yq
